# 기업 제출용 포트폴리오 개편 설계

## 목적
현대오토에버 Embedded SW QA Engineer 지원을 위해 기존 개인 브랜딩형 페이지를 기업 제출용 Executive Brief 포트폴리오로 개편한다. 채용담당자가 짧은 시간 안에 전공, 직무 역량, 대표 프로젝트, 자격 및 수상 근거를 확인할 수 있어야 한다.

## 정보 구조
1. **Hero**: `조정빈 | Embedded SW QA Engineer`, 국민대학교 자동차IT융합학과 졸업, CANoe/CAPL 기반 차량 SW 테스트 설계와 자동화 검증 수행 경험을 2문장 이내로 제시한다.
2. **핵심 요약**: `국민대학교 자동차IT융합학과 졸업`, `ISTQB CTFL`, `Black Box Testing 프로젝트 우수상`을 첫 화면에 노출한다. `812시간`은 교육사항으로 이동한다.
3. **핵심 역량**: Test Design, Test Automation, Defect Analysis, Embedded SW Understanding의 4개 업무 단위로 정리한다.
4. **대표 프로젝트**: CANoe/CAPL 기반 Black-Box Validation 프로젝트를 목표, 역할, 수행, 결과 구조로 요약한다. 장식용 그래프보다 실제 수행 내용과 수상 성과를 우선한다.
5. **교육사항**: HL만도·HL클레무브 IVS 5기 812시간, A-SPICE 단과과정 및 취업특강을 별도 영역에 배치한다.
6. **자격·수상**: ISTQB CTFL, 정보처리기사, Black Box Testing 프로젝트 우수상, IVS 5기 모범상을 간결한 이력형으로 배치한다.

## 핵심 문구
> 국민대학교 자동차IT융합학과에서 자동차·전자·소프트웨어 융합 지식을 쌓고, CANoe/CAPL 기반 차량 SW 테스트 설계와 자동화 검증을 직접 수행했습니다.

`분석해 왔습니다` 같은 경력 누적형 표현은 사용하지 않는다. 신입 지원자에게 적합한 `직접 수행했습니다`, `경험했습니다`, `구현했습니다` 중심으로 작성한다.
